# Pages OAuth - Kreolyse

Ce dossier contient les pages statiques pour la validation OAuth Google.

## Fichiers

- `index.html`: page d'accueil de l'application
- `privacy.html`: politique de confidentialite
- `terms.html`: conditions d'utilisation
- `styles.css`: style partage

## Publication sur GitHub Pages

1. Creer un nouveau repo public (ex: `7aidproductionsweb.github.io`).
2. Copier le contenu de ce dossier a la racine du repo.
3. Activer GitHub Pages sur la branche principale.
4. Verifier que les URLs sont accessibles publiquement:
   - `https://7aidproductionsweb.github.io/`
   - `https://7aidproductionsweb.github.io/privacy.html`
   - `https://7aidproductionsweb.github.io/terms.html`

## Parametres OAuth Google

- Homepage URL: `https://7aidproductionsweb.github.io/`
- Privacy policy URL: `https://7aidproductionsweb.github.io/privacy.html`
- Terms of service URL: `https://7aidproductionsweb.github.io/terms.html`
- Authorized domain: `7aidproductionsweb.github.io`

## A personnaliser avant soumission

- Remplacer `[A RENSEIGNER: email ou formulaire de contact]` dans `index.html`.
